Only show products in specific category. It will only show products from main category instead of showing all products from all sub categories. Generally woocommerce shows all products from main and sub categories.

== Description ==

By using this plugin you can hide products of sub categories. This plugin will allow only related products of specific category either from sub categories or main categories.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `hide-sub-categories-products` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto settings and you will find plugin there.

= Manual installation =

**Uploading in WordPress Dashboard**

1. Download `Hide sub categories products`
2. Navigate to the 'Add New' in the plugins dashboard
3. Navigate to the 'Upload' area
4. Select `hide-sub-categories-products.zip` from your computer
5. Click 'Install Now'
6. Activate the plugin in the Plugin dashboard
7. Find it under Settings menu

**Using FTP**

1. Download `hide-sub-categories-products.zip`
2. Extract the `hide-sub-categories-products.zip` directory to your computer
3. Upload the `hide-sub-categories-products.zip` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard

The WordPress codex contains [instructions on how to install a WordPress plugin](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
